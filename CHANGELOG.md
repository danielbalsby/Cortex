# Changelog

## 0.4.0-alpha

### Added
- Sticky consultation cockpit
- Foldable clinical cards
- Clinical review located next to generated output
- Administrative output overview
- Persistent copy action
- Expanded assessment suggestions
- Journal, X-ray and orthopaedic referral outputs

### Known limitations
- Knee workflow requires further clinical validation
- Physiotherapy referral is not implemented yet
- macOS distribution is experimental
- No EHR integration
- No storage of patient encounters
