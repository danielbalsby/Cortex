# Cortex v0.1

## Formål
Cortex skal gøre det hurtigere at finde og kopiere kliniske standardformuleringer uden at afbryde konsultationens flow.

## Første vertikale slice
1. Søg i titel, kategori, tags og tekst.
2. Navigér mellem resultater med tastatur.
3. Se hele frasen i et roligt preview.
4. Kopiér med Enter eller én knap.
5. Fastgør hyppigt anvendte fraser lokalt.

## Ikke inkluderet endnu
- Patientdata
- Cloud og login
- Smart Phrase-builder
- Journalsystemintegration
- Klinisk beslutningsstøtte
- AI-generering

## Succeskriterier
- Relevant frase findes på under fem sekunder.
- Kopiering kræver ingen sideændring.
- UI fungerer på laptop og mobil.
- Appen kan bruges uden patientidentificerbare oplysninger.
