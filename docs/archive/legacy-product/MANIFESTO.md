# Cortex Manifesto

We build software that gives clinicians more time with patients.

We document each clinical fact once and reuse it safely.

We design workflows before features.

We prefer calm, predictable interfaces over flexibility and visual noise.

We automate administration, never clinical responsibility.

Every recommendation must be understandable and traceable.

Every new function must make the consultation faster, simpler or safer.
