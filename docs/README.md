# Cortex documentation

The canonical foundation is in [`vision/`](vision/README.md). Read it before making product, clinical, design or architectural decisions.

- [`governance/`](governance/) contains engineering and clinical safety constraints.
- [`product/`](product/) contains the current [roadmap](product/ROADMAP.md), requirements and proposals.
- [`architecture/`](architecture/) describes the current system, proposed RFCs and accepted decisions.
- [`clinical/`](clinical/) contains clinical pathway documentation.
- [`design/`](design/) contains current [UX guidance](design/UX-GUIDELINES.md), the [clinical document workspace model](design/CLINICAL-DOCUMENT-WORKSPACE-v1.md) and wireframes.
- [`archive/`](archive/README.md) contains read-only historical records.

Archived documents are not authoritative.
